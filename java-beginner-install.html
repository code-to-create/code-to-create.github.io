Java - Beginner

Welcome to Java!

This course will give you a comprehensive introduction to the Java programming language: a useful, versatile, and popular language which can be used to create cool products such as Android and Web Applications. This is an introductory course, meaning no prior programming experience is required; however, a basic understanding of computer science principles is recommended. Since this will be many students' first experience with text-based programming, this course will focus on Java fundamentals, such as variables, operators, control statements, etc. Java is a vast language, so more advanced topics, such as algorithms, data structures, and graphics, will be discussed in later Java courses.

Java setup

Because of the relative complexity in compiling and running Java files, many programmers use an Integrated Development Environment (IDE) to edit and run Java programs. We will use Eclipse, a free IDE that can be downloaded from eclipse.org.

Install Java 8

Eclipse requires Java 8, which you can download from Oracle. You must first accept their license agreement, after which you can download the installer for your operating system.

Mac OSX

The downloaded file will open a disk image, and you will have to double click the .pkg file inside the disk image to run the installer. You will need an account with administrator privileges to install Java 8.

Windows

On some Windows computers, the JDK 8 installer will not properly configure your computer to work with Java. This can usually be fixed by setting your PATH variable.

Another possible issue that arises on Windows is when there are multiple versions of Java installed. Try uninstalling all versions of Java from the Control Panel, then re-install Java 8.

Install Eclipse

The Eclipse installer can be downloaded from eclipse.org/downloads. Select your operating system to download its Eclipse installer.

Click the orange download button to start downloading Eclipse.

Open the downloaded Eclipse Installer and click "Eclipse IDE for Java Developers".

You can install Eclipse in the default eclipse folder, or select a different folder.

On Mac OSX, you may want to change the installation folder to /Applications/eclipse if you don't plan on installing multiple versions of Eclipse.

Accept the Eclipse license agreement (or click "Decide Later") to begin the installation.

Once Eclipse has completed installation, you can click the green Launch button to open it.

How to use Eclipse

Eclipse can be used with many programming languages, and has an immense set of features and tools under the hood. In our course, we will only need to do three basic things with Eclipse - create folders, create files in those folders, and import code from external files.

When you open Eclipse for the first time, you will need to click the "Workbench" link.

The Workbench is just a fancy text editor that starts off with a lot of additional panels that we won't need. Click the x's to close every panel except for the Package Explorer.

You can re-open any panel by going to Window > Show View and clicking on the name of the panel.

Create a project

A Java Project is just a folder on your computer that is managed by Eclipse. You can create Java Projects by right clicking on the Package Explorer, hovering over "New", and selecting "Java Project".

If you hid the package explorer, navigate in the top menu to Window > Show View > Package Explorer to re-open it.

In the dialog box that pops up, enter a name for the project, then click "Finish".

Create a file

To create a file, right click on the src folder, hover over "New", and select "Class".

Create a package

A package is a sub-folder within the Java Project that provides an additional level of organization for your files.

Vocabulary

Operating system - The program that manages the underlying operations of your computer. This is usually Mac OSX (for Apple computers), Windows, or Linux.

Integrated Development Environment - A program that makes it easier for a programmer to edit and run programs, commonly abbreviated as IDE.

Eclipse - A free Integrated Development Environment used in Java development.

Java 8 - The latest version of the Java programming language.

Build path - The folders on your computer where Java looks for libraries and external resources.
