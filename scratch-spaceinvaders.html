Lesson 3: Space Invaders Exercise

Introduction to Space Invaders

Now that you have more experience with Scratch, we will be building more complex games with more challenging instructions! In the previous lessons, we learned how to create scoring based on the interaction between objects, how to dictate movement through the use of a cartesian coordinate system, how to personalize, and how to create losing and winning scenarios

In this exercise, instructions will be given “abstractly”. This means instead of giving instructions that are step by step oriented, we will give more general instructions. You may be confused, but let us begin building one of the most influential games of all time.

Space Ship Basic Movement

The space ship should only move along the x axis yet maintain a fixed y negative value. Set the controls of the spaceship to the left and right key pads. Every time you hit the left or right arrow key, the ship should move x spaces.

Alien Movement and Cloning

New Concept: Cloning

Cloning is a command script allowing a sprite to clone or duplicate itself, while the project is running. Clones will be the same as the original but as a separate entity. Clones can have the same features, but can then be modified. Notice that there are three blocks related to cloning: "Create a clone of ()" , "When I start as a Clone", "Delete this Clone".

Build two types of aliens that will descend to the bottom of the screen. They should differ by speed at which the ball and their starting position should be randomized. (Hint: Unlike the starship, the aliens do not have a fixed y-value)

Laser Construction

The behavior of the laser will have two basic command blocks. A third can be added for stylization and graphics. Correspond the laser firing when the space key is pressed. (Hint: Use a create cloning script within the forever/if space key statement). If the laser touches the edge, then it should delete itself and if the laser touches either alien invader, then the laser should also delete itself (Hint: If touching or or _ )

Scoring

Create a scoring variable. This variable will be set to zero when the game begins. A positive point will be awarded for every time the laser touches an alien. Every time the laser touches an alien clone, both the laser and alien clone should delete itself. If an alien passes the bottom of the screen( a certain value of y), then the score changes by x points. If you have more than 15 points, then you lose. If you have more than 40 points, then you win the game. If the alien touches the star fighter, then it the star fighter will automatically lose.
