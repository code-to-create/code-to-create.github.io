Scratch - a visual introduction to programming

Scratch is a drag-and-drop programming environment created by the MIT Media Lab. Scratch is a block-based environment, meaning that it uses color coded blocks which can be dragged, dropped, and fitted together to represent statements and create programs. It makes computer science education easily accessible to all ages and skill levels through an intuitive interface that provides immediate visual response.

This course is recommended to everyone, regardless of experience.
